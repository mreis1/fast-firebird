# Brand assets

- `logo.svg` — horizontal wordmark lockup (README banner, docs).
- `mark.svg` — the firebird mark alone (app header, favicon source).
- `favicon.ico` — multi-size icon (16/32/48/64/256), for browsers/OSes without SVG favicon support.
- `png/` — raster exports of the mark (16–512 px, transparent) plus `logo-1440x300.png` (wordmark, e.g. social/preview cards).

The mark is a firebird in flight — flame gradient, head/beak/eye, speed streaks —
conveying speed. Regenerate rasters from the SVGs with the canvas rasterizer
(see diary 2026-07-09, session 12) or any SVG→PNG tool.
