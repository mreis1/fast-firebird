## Decisions
- Wire protocol floor = protocol 13 (FB3). FB 2.5 out of scope for now.
- Pure TS, node:crypto only (BigInt SRP, rc4 via createCipheriv, node:zlib).
- Jaybird-style layering with per-protocol-version strategy; deferred packet queue
  for round-trip elimination.
- Docker: compose project `fast-firebird-test`, ports 30503/30504/30505, cleanup
  only via scoped script. Never any prune/global command.
